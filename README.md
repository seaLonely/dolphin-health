## 🛠️ 技术栈

- **Flutter 3.x** - 跨平台框架
- **SQLite** - 本地数据库
- **Provider** - 状态管理
- **fl_chart** - 图表库
- **intl** - 国际化
- **share_plus** - 分享功能
- **path_provider** - 文件路径
